## Important

- **Do not rename** this repository or the GitHub account. All NFC tags and printed QR codes contain hardcoded URLs. A rename breaks every deployed tag.
- **Do not use `git push`** from a local machine. Always use the GitHub web editor to prevent email leakage in commit metadata.
- **Do not add** real names, personal email addresses, or other identifying information to any file in this repository.
